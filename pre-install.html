PCAT Tool

燒錄 image 工具Image flashing tool
  1. 打開 Qualcomm Software Center。Open Qualcomm Software Center.
  2. 在 Tools 搜尋 Product Configuration Assistant Tool，下載並安裝。Under Tools, search Product Configuration Assistant Tool, then download and install.
03

QUD (Qualcomm USB Driver)

Qualcomm USB 驅動Qualcomm USB driver
  1. 打開 Qualcomm Software Center。Open Qualcomm Software Center.
  2. 在 Tools 搜尋 Qualcomm USB Driver，下載並安裝。Under Tools, search Qualcomm USB Driver, then download and install.
小提醒:一次只能安裝一個工具，若畫面顯示 No logs found 訊息，代表該工具尚未安裝。 Note: Please install only one tool at a time. If the No logs found message appears, it indicates that the tool has not been successfully installed.
04

Model Conversion Env Setup (For iQ-foundry)

安裝環境為當日實作(Hands-on的基礎)，請務必於課前完成配置。 A pre-configured environment is essential for the hands-on session; please ensure the installation is successful beforehand.
  1. 下載到您指定的資料夾https://github.com/InnoIPA/iQ-Foundry/blob/main/setup-windows-wsl.ps1 Download to your specified folder.https://github.com/InnoIPA/iQ-Foundry/blob/main/setup-windows-wsl.ps1
  2. 以管理員權限開啟 Windows PowerShell Open Windows PowerShell with Administrator rights:
    bash 
    cd "$env:USERPROFILE"
  3. 在PowerShell開啟腳本所在的資料夾，執行以下指令，此步驟通常需要 5-10 分鐘。 Open the folder where the script is located in PowerShell and execute the following command.This step typically takes 5-10 minutes.
    bash 
    powershell -ExecutionPolicy Bypass -File .\setup-windows-wsl.ps1
若安裝過程中遇到權限限制，請聯繫貴公司的 IT 或 MIS 部門協助排除。 After installing, QSC showing you have no devices connected is normal (no board attached yet).

02

QDL (命令列燒錄工具CLI flashing tool)

命令列燒錄工具CLI flashing tool
  1. 安裝相依套件：Install dependencies:
bash 
sudo apt install -y git libxml2-dev libusb-1.0-0-dev libzip-dev meson ninja-build help2man
  1. 編譯並安裝 qdl：Build and install qdl:
bash 
git clone --depth 1 --branch master https://github.com/linux-msm/qdl
cd qdl
meson setup build
meson compile -C build
sudo meson install -C build
  1. 驗證安裝：Verify the install:
bash 
qdl --help
03

Docker

容器執行環境Container runtime
  1. 更新系統：Update the system:
bash 
sudo apt update && sudo apt upgrade -y
  1. 下載 Docker 官方安裝腳本：Download the official Docker install script:
bash 
curl -fsSL https://get.docker.com -o get-docker.sh
  1. 執行安裝腳本：Run the install script:
bash 
sudo sh get-docker.sh
  1. 將使用者加入 docker 群組（需重新登入生效）：Add your user to the docker group (re-login to take effect):
bash 
sudo usermod -aG docker $USER
  1. 啟用並啟動 Docker 服務：Enable and start the Docker service:
bash 
sudo systemctl enable --now docker
  1. 驗證安裝：Verify the install:
bash 
sudo docker run hello-world
官方安裝文件：https://docs.docker.com/engine/install/ubuntu/ Official install docs: https://docs.docker.com/engine/install/ubuntu/
04

minicom (UART 序列埠工具UART serial tool)

看 boot log / debugView boot log / debug
  1. 安裝 minicom：Install minicom:
bash 
sudo apt install -y minicom
  1. 連線（CP210x 通常為 /dev/ttyUSB0，鮑率 115200）：Connect (CP210x is usually /dev/ttyUSB0, baud rate 115200):
bash 
sudo minicom -D /dev/ttyUSB0 -b 115200
