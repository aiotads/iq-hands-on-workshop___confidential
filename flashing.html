燒錄說明書 Windows
Flashing Guide

本頁示範用 xPCATApp 將映像燒錄到 Q911 平台。開始前請先完成「預先安裝」中的 QSC、PCAT、QUD。 This page shows how to flash the image to the Q911 platform with xPCATApp. Finish QSC, PCAT, and QUD in "Pre-install" first.
燒錄異常排除： 1. 拔掉電源線 2. 重新上電 3. 再次燒錄。若需要協助，請洽現場助教。 Flashing Troubleshooting: 1. Unplug the power cable. 2. Replug to power cycle. 3. Retry flashing. You can also ask our TAs for help.
燒錄步驟Flashing steps
1 前置步驟Prep 2 xPCATApp 3 連線 Q911Connect Q911 4 Download 5 選 image / UFSImage / UFS 6 完成燒錄Done 7 切回 NormalNormal mode 8 啟動進入系統Boot into the System
1

準備電路板Prepare the Board for Flashing

請確保裝置已拔掉電源，才能執行以下步驟喔! You must unplug the power supply before proceeding with the following steps!

請將板子背面的跳帽調整到 EDL mode。 Set the jumper on the bottom side of the board to EDL mode.

同時也請確認所有的啟動模式撥碼開關（DIP switches）皆已調至 ON（如下圖所示），系統才能從 UFS啟動。 Please also confirm that all boot mode DIP switches are set to ON, as shown below, so the system can boot from UFS.

請將 USB Type-C 傳輸線連接到目標裝置上標示為 Flash / ADB的連接埠。並接上電源，接完後請開機。 Connect the USB Type-C cable to the port labeled Flash / ADBon the target device.Plug in the power, and then turn it on.

2

打開 xPCATAppOpen xPCATApp

請確保裝置已開機，才能找到設備喔! Please make sure the device ispowered on so it can be detected!

打開 xPCATApp。 Open xPCATApp.

請確認是否開啟 "x"PCATApp Please confirm whether"x"PCATAppis open or not.
3

連線 Q911Connect to the Q911

點擊Icon，並選擇要連線的 Q911。 Click the icon and choose Q911 to connect.

4

點擊 download 選項Click the download option

點擊左側的 download 選項。 Click the download option on the left.

5

選擇 image 與 memory typeChoose image and memory type

選擇 image 所在的資料夾和選擇 UFS 作為 memory type，點擊 download。 Select the folder containing the image, choose UFS as the memory type, and click download.

請點擊 ∨ ，並在下拉選單中選擇要Folder選項，請選擇 workshop_exmp-q911_v2.3.3_image。 Click ∨ to open the drop-down menu, then select workshop_exmp-q911_v2.3.3_imagefor the Folder option.
6

完成燒錄Flashing complete

完成燒錄。 Flashing complete.

7

拔掉電源後切回 Normal mode，並開機Power off and unplug the device, switch to Normal mode, then turn it on.

請確保裝置已拔掉電源，才能執行以下步驟喔! You must unplug the power supply before proceeding with the following steps!

進行下方的步驟 (切回 Normal mode 並開機）。 Proceed with the steps below (Switch back to Normal mode and boot).

燒錄成功後，請將裝置斷電，並把板子背面的跳帽調回 Normal mode。 After the flashing process is complete, power off the device and set the jumper on the bottom side of the board to Normal mode.

請一併確認所有的啟動模式 DIP 指撥開關都已經切換到 ON，以確保系統從UFS啟動。 Please also confirm that all boot mode DIP switches are set to ON, as shown below, so the system boots from UFS .

8

接上螢幕、滑鼠，啟動進入系統Boot into the System

切換回 Normal 模式後，接上螢幕、滑鼠以及電源即可開啟裝置電源並啟動進入系統。 After switching back to Normal mode, connect the monitor, mouse, and power supply to power on the device and boot into the system.

點擊左上方紅色ICON如果出現版本號表示燒錄成功(太棒了🥳)。 Click the red icon in the top-left corner, and if the version number and successful flashing message appear, the system has booted correctly.🥳
